GS2-Obfuscator is a HTML/JavaScript tool designed to obfuscate and deobfuscate GraalScript 2 code by encoding strings, properties, and numbers using Base64 encoding. It helps obscure variable names and values, making the code harder to understand at first glance. The tool supports recursive property decoding, ensuring that nested properties are correctly handled. It also includes features for encoding numeric values with @float() wrappers and provides a simple API for both obfuscation and deobfuscation.
